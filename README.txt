Movie trailer website

What is this?
-------------

This is a project created for the Udacity Full-Stack Webdeveloper nanodegree
program.  When run, it will open a webbrowser to display a number of movie
poster images, each of which can be clicked on to open the movie's trailer.

How to run
----------

The user should run the entertainment_center.py file using Python 2.7.  All
other files included with this project must be located in the same directory
to ensure proper behavior.
